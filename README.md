# Projet Python+PL

Optimisation de la gestion des déchets dans une ville (ville de Marrakech)
Maximiser l'efficacité des camions
Maximiser la quantité des déchets collectées (avec le minimum des camions)


## Variable de décision
- x1: nombre de camions destinés pour la collecte des déchets ménagées
- x2: nombre de camions destinés pour la collecte des déchets industriels


## Fonction Objectif
Min Z = x1 + x2


## Contraintes
- Quantité de déchets à collecter: 2 x1 + 3 x2 >= 20
- Nombre de camions limités: x1 + x2 <= 10
